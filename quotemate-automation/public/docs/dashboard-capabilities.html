RAG Context: Vector Search Over Past Quotes

Every intake is embedded with Voyage embeddings and matched against prior jobs using Supabase pgvector's match_intakes function. The top-k similar past quotes are fed back to Claude as in-context examples for sharper pricing on the new job — a learned-from-experience effect with no fine-tuning required.

Live

Grounding Gate (Safety Check)

Before any quote ships, a grounding gate verifies that every line-item price derives from the tenant's pricing book and assembly catalogue scoped to the correct trade. Any line that can't be grounded downgrades the entire quote to the $99 inspection route — the safe failure mode.

Live

Claude-Drafted Quotes (Anthropic Flagship)

Quotes are drafted by Claude Opus 4.7 with the SMS dialog, slot extraction, and intent classification on Sonnet 4.6 for higher accuracy on conversational nuance. The system prompt is loaded server-side with deterministic tool definitions for material/labour lookups.

Live

Anthropic Prompt Caching

The estimator system prompt — pricing book, assembly catalogue, trade rules, examples — is cached server-side using Anthropic's prompt caching, so per-quote latency and token spend drop dramatically after the first hit. Critical for unit economics at scale.
